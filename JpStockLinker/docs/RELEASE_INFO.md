## 📋 Google Play Console 登録情報

### アプリ説明（短い説明 - 80文字以内）
```
日本株の情報サイトへワンタップでアクセス。お気に入り銘柄を管理、ウィジェット対応。
```

### アプリ説明（詳細説明）
```
JpStockLinkerは、日本株投資家のための銘柄管理＆リンク集アプリです。

【主な機能】
📊 銘柄管理
• 銘柄コードと名前を登録してマイリストを作成
• お気に入り機能で重要銘柄をマーク
• グループ分け（監視中・保有中・検討中・売却済など）
• メモ機能で目標株価や損切りラインを記録

🔗 ワンタップリンク
• Yahoo!ファイナンス、株探、TradingView、みんかぶなど10以上のサイトに即アクセス
• 銘柄を選択してボタンを押すだけ

📱 ホーム画面ウィジェット
• お気に入り銘柄をホーム画面に表示
• ウィジェットから直接各サイトへジャンプ
• 株価を自動取得して表示

🔍 高速検索
• 4000銘柄以上のマスターデータ搭載
• コード・名前・セクター・テーマで検索

【対応サイト】
Yahoo!ファイナンス / 株探 / TradingView / みんかぶ / IR BANK / 
四季報オンライン / バフェットコード / 日経 / 空売りネット など

【特徴】
✓ 広告なし
✓ ユーザー登録不要
✓ データは端末内のみに保存（プライバシー保護）
✓ 折りたたみスマホ対応
```
